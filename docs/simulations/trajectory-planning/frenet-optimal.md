# Frenet Optimal Trajectory

## Algorithm

Plans in Frenet coordinates (along-track, cross-track) with quintic lateral and quartic longitudinal polynomials, selecting the minimum-cost candidate.

**Reference:** M. Werling et al., "Optimal Trajectory Generation for Dynamic Street Scenarios in a Frenet Frame," ICRA, 2010.

## Run

```bash
python -m uav_sim.simulations.trajectory_planning.frenet_optimal
```

# Occupancy Mapping

## Algorithm

Probabilistic grid mapping using log-odds updates from simulated lidar. Visualised as a 3D voxel scatter of occupied cells.

**Reference:** S. Thrun, W. Burgard, D. Fox, "Probabilistic Robotics," MIT Press, 2005, Ch. 9.

## Run

```bash
python -m uav_sim.simulations.perception.occupancy_mapping
```

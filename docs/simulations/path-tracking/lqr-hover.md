# LQR Hover

## Algorithm

Continuous-time Linear Quadratic Regulator linearised about the hover equilibrium. Minimises a quadratic cost on state deviation and control effort.

**Reference:** B. D. O. Anderson, J. B. Moore, "Optimal Control: Linear Quadratic Methods," Prentice-Hall, 1990.

## Run

```bash
python -m uav_sim.simulations.path_tracking.lqr_hover
```

# Leader-Follower

## Algorithm

A leader traces a circular path while followers maintain desired offsets using PD control on formation error.

**Reference:** J. Desai et al., "Modeling and Control of Formations of Nonholonomic Mobile Robots," IEEE T-RA, 2001.

## Run

```bash
python -m uav_sim.simulations.swarm.leader_follower
```

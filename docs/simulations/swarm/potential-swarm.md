# Potential-Based Swarm

## Algorithm

Agents navigate to a goal using Lennard-Jones inter-agent potentials for collision avoidance and formation maintenance, plus a goal-attraction force.

**Reference:** W. M. Spears et al., "Distributed, Physics-Based Control of Swarms of Vehicles," Autonomous Robots, 2004.

## Run

```bash
python -m uav_sim.simulations.swarm.potential_swarm
```
